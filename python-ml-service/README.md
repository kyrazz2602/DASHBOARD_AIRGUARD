# Python ML Service

FastAPI microservice that hosts the trained ML model for air filter condition estimation.

## Setup

### 1. Place Model Files

Copy your trained model files into the `models/` directory:

```
python-ml-service/
└── models/
    ├── best_model.pkl   # Trained classifier (Random Forest / Decision Tree / SVM)
    └── scaler.pkl       # StandardScaler fitted during training
```

These files are generated by your ML training pipeline. The service will fail to start if either file is missing.

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Service

```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```

For development with auto-reload:

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## Endpoints

- `POST /predict` — Accepts `{ pm25, pm10, co, voc, suhu }` and returns filter status prediction with probabilities and recommendation.
- `GET /health` — Returns `{ "status": "ok" }` when the service is running.

## Environment

Set `PYTHON_ML_SERVICE_URL=http://localhost:8000` in your Next.js `.env.local` to connect the frontend to this service.
